# Sistema de Mantenimiento CSL / DEPICENTER

## Para correr localmente:
1. Instalar Node.js desde nodejs.org (botón LTS)
2. Abrir cmd y ejecutar: npm install -g pnpm
3. Entrar a la carpeta: cd C:\csl-app
4. Instalar dependencias: pnpm install
5. Iniciar: pnpm dev
6. Abrir navegador en: http://localhost:3000

## Para publicar en Vercel:
1. Subir esta carpeta a GitHub
2. Conectar el repositorio en vercel.com
3. Click Deploy

## URL del sistema publicado:
https://mantenimiento-csl.vercel.app
